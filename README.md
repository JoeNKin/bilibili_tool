### 总功能

1:b站获取cookie 

2:wb获取cookie 

3:b站获取关注列表

4:人工校验ck 

5:清除没有消息的b站关注 

6\.剪贴板b站uid提取 

7\.b站浏览器端js脚本

### b站js脚本（控制台中输入）

1:提取网页源代码中的所有链接
2:提取网页源代码中的所有链接（哪个好用哪个效果不同页面效果不同)
3:提取剪贴板中的所有链接
4:每2.2秒删除一天动态（速度懂者可以自己调）
5:每2.2秒删除一天动态自带滚落（推荐这个）
6\.移除不要删除动态（辅助4，5）
7\.取关

tips：按f12开启控制台
